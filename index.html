    Software Development & Programming Languages Skills

    HTML5/CSS3

    As this description follows the chronological introduction I have had to various technologies, I am now returning to languages used in software development. HTML 5 and CSS 3 are used in conjunction with one another so thoroughly that it is not useful to separate them from one another in this context. This skill set is probably the only one I am more comfortable with than even Linux as it is one that I mastered since devoting myself completely to tech, which began about 2 years ago and has remained consistent ever since. While not terribly complex languages to learn, their mastery does take either a significant dive into the subject independently and avoiding the tendency that many have of learning pieces of them in conjunction with JS as doing so leaves too much of the underlying logic unsaid.

    I have also received a Front End Certificate that took a significant dive into these tandem languages, JS, libraries of JS and APIs that helped to deepen my appreciation for the use of these languages in practice and one class I am in currently is on them specifically. While not requiring the sheer technical know-how or analytical capacity of JS, they speak to another skill set I bring with me (but will not presently discuss ad nasesum) which is design and thus creative capacity of a more detailed depth than imagining if pushing and popping elements out of an array will accomplish the task as there is little easily articulated in the creative with the words present in the English lexicon. Being recent, I do have **some** of the decent examples of my endeavors of these languages posted on any of my 3 VC accounts (Bitbucket, GitLab, GitHub) but being a one man development team mastering React, I don't at present have the time to go through and edit them all for placement in the public and personally don't release low-quality code to the public due to quirks of my character.

    Javascript ES05 // JQuery // Bootstrap // Vue

    Javascript, which is also sometimes called JS, vanilla JS etc, is the scripting language that emerged from NetScape and took over the internet. As with anyone in tech today, I could not avoid JS if I hope to be a useful memeber of any modern technology development outfit for the client side rendering that this language enables is such that the browser can perform some of the compute tasks that would otherwise have been done on a server thus leveraging user systems to lower overhead and infrastructure complexity. I also happen to like JS as a programming language simply because it is a scripting language and demonstrates it in ways that are more apparent to me to be descendant from the BASH Shell.
